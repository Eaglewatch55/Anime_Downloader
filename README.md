# Anime_Downloader

# --- Script created for personal use.

Requirements:
- Python 3.10

Python Libraries:
- requests_html
- tkinter
- os
- time
- tinydb
- pysftp

It scraps the animeflv´s anime chapter page, and Downloads the next episode on demand according the record ont the script´s database.
Its capable to add and remove the shows to scan.
The GUI has a log to see the status of the procces, and if its succesfull or not.
Its capable to send the file via sftp to another device.
